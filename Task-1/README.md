# Task 1 - Titanic Survival Prediction

The goal is to predict whether a passenger on the Titanic survived or not using machine learning.

## Files inside Task-1

- data/ → Titanic-Dataset.csv
- models/ → final_model.pkl
- src/
  - train_model.py
  - predict.py
  - preprocess.py
  - evaluate.py
- requirements.txt

## How to Run

1. Install packages:
   pip install -r requirements.txt

2. Train the model:
   python src/train_model.py

3. Run prediction:
   python src/predict.py
